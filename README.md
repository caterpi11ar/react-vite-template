A React + Vite template with TanStack Router, TanStack Query, Zustand and Tailwind CSS v4 preconfigured.

## Features

- [x] **React 19** with React Compiler
- [x] **Vite 8** (rolldown)
- [x] **TypeScript** strict mode
- [x] **TanStack Router** — file-based routing
- [x] **TanStack Query**
- [x] **Tailwind CSS v4**
- [x] **Zustand**
- [x] **Axios**
- [x] **ESLint** (`@antfu/eslint-config`)
- [x] **Husky + lint-staged + commitlint**
- [x] **pnpm-only**

## Quick Start

```bash
pnpm install
pnpm dev
```
